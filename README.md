# InterViewCoding

**Purpose:** Hands-on Java practice for data structures, algorithms, and typical interview-style problems—built and maintained as a public portfolio of consistent coding work.

**For hiring teams:** This repo shows deliberate preparation: working implementations (not snippets), progression from fundamentals (arrays, strings, recursion) to structures like linked lists, plus runnable time-complexity examples. Supporting notes and problem lists live under `resources/` for depth; the root holds executable Java sources you can review or run.

**Stack:** Java (JDK 8+).

**Quick run** (from repo root):

```bash
javac YourFile.java
java YourFile
```

**Structure (high level):**

| Area | What to look at |
|------|------------------|
| Problems & drills | `TwoSum.java`, `Palindrome*.java`, `ReverseArr.java`, `MissingNumber.java`, `Duplicate.java`, and related files |
| Linked lists | `LinkedListPractice.java`, `SimpleLinkedList.java`, etc. |
| Complexity | `*Time.java`, `TimeComplexityDemo.java`, `README_TimeComplexity.java` |
| Study material | `resources/` (guides, checklists, curated problem lists) |

If you need a signal in one line: **active interview prep in Java, versioned on GitHub, with both code and structured learning assets.**
